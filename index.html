1. Na tag input com atributo type="text", para que serve o atributo "placeholder"?
    • O atributo placeholder é uma pequena frase ou palavra, que tem o intuito de ajudar usuários a entender como devem preencher determinado formulário, inserindo um texto contextual no campo a ser respondido.
  2. Na tag input com atributo type="radio" ou type="radio", para que serve o atributo "checked"?
    • CHECKED: especifica que este botão "checkbox" ou "radio" é selecionado por default.
  3. Na tag input para que serve o atributo "value"?
    • Definir o valor a ser enviado por este item.
  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
    • Quando tem o name o mesmo do anterior
  5. Para que serve a tag label? Para que serve o atributo "for" nessa tag?
    • Se associada a um elemento do formulário permite que você "clique" nesse elemento clicando em sua label. Essa funcionalidade é especialmente útil se for usada em checboxes , botões e radio buttons .
  6. Na tag option para que serve o atributo "selected"?
    • Este atributo booleano indica que várias opções podem ser selecionadas na lista. Se não for especificado, apenas uma opção poderá ser selecionada de cada vez.
  7. Nas tags de formulário para que serve o atributo "required"?
    • Esse é um atributo booleano usado para indicar que um determinando campo de formulário é obrigatório para o envio do mesmo. Ao adicionar esse atributo a um campo de formulário, o navegador obriga o usuário a inserir dados naquele campo antes de enviar o formulário.
  8. Na tag form para que serve o atributo "action"?
    • O atributo action define o local (através de uma URL) ao qual serão enviados todos os dados recolhidos do formulário.
  9. Na tag form para que serve o atributo "method"?
    • O atributo method define qual o método HTTP para enviar os dados (ele pode ser "GET" ou "POST"
  10. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?
    • Para que não apareça na url o elemento digitado "Post". Obs: Nunca use GET para enviar dados confidenciais! (será visível no URL)
  11. Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?
    • Será o valor "Post" pois ele: Anexa dados de formulário dentro do corpo da solicitação HTTP (os dados não são mostrados na URL) Não tem limitações de tamanho Envios de formulários com POST não podem ser marcados.
